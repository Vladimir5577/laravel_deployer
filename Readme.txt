Install laravel 8 you need PHP >= 7.3

	$ composer create-project --prefer-dist laravel/laravel blog

Go to the folder and install sleepingowl

	$ cd <folder name>

	$ composer require laravelrus/sleepingowl

In .env file put your credentials  to database

Run project in the terminal type
	
	$ php artisan serve

In browser type

	localhost:8000
